当“白嫖”不再是一种习惯
发表于 2020-08-19 22:37:34| 更新于 2020-09-07 22:26:23|一般文字杂文随笔
字数总计:1.1k|阅读时长: 3 分钟

  现在提起“白嫖”这个词，已经不是本来简单的“吃霸王餐”了，引申起来大概是：“不花费任何资源去进行某件事情”，“免费索取他人资源的行为”。

  说起来自己也是老“白嫖党”了。老些年无论是看动漫、电视剧，还是下单机游戏，甚至脑抽找些教学资料，第一时间都是去资源聚合网站、各种论坛去找。说起来当年要白嫖也是有苦衷。去贴吧之类的找漫画、动画、影视剧，熟肉来得快，不用强制看广告，下载到本地想怎么看都行，尤其在想看片但室友又在玩网游的时候。玩单机游戏更是没有概念，原来单机游戏是要钱的啊，我看都是什么绿色版、免安装版，还以为本来就是不要钱的呢。那个时候，“楼主好人”、“好人一生平安”倒也是别人才会打出来，我也只是默默点开链接，除非“回复可见”，那我只能关闭网页。

  不过现在至少网络带宽比六七年前好很多了，所以视频之类的，下载本地看还是在线看，进度条随意拖动等待时间也不会太长了。尤其在国家相关政策下，版权意识、正版意识，也普遍提高了很多，现在音乐、视频等聚合资源网站，动不动就是几个亿几十个亿签约、引入、合作等，大概也会让人意识到，原来他们的歌曲、影视，也是有成本的啊，不是大风挂的啊。

  之后随着互联网发展，短视频、网红直播，倒是一大票流量，流量从根本角度来说，那就是钱啊。所以自制短视频也好、文章也好，大抵都有点赞、收藏等功能，如果创作者为平台带来足够的流量，那平台的支持也是重金投入了。

  可惜，对于个人来讲，不靠他成家立业，似乎并没有多少相干。例如 B 站视频本来一直有点赞、投币、收藏按钮来支持视频创作者，那某些程度是可以为创作者带去经济利益，然后以期产出更优秀的作品。我记忆中，6、7 年前我有开始从 B 站爬取各种视频资源，尤其是有些冷门没有熟肉的时候。后来工作了不好再用别人的账号，于是自己注册了一个。得亏我昨天看了半个视频，今天想继续看时找不到历史记录，我怀疑我还是 0 级会员没有转正平台没真没认真留存。一千多天之后，我才答题转正，成为 1 级会员，然后我才发现我有一枚硬币了。

  每天大概要花了 2、3 个小时在 B 站刷视频，再多精彩的，也就只是看一看，从未三连，其实也好说，我都没有转正，只是注册了个账号，只图看视频，不作评论，不发弹幕，简单白嫖。

  人是会变的，尤其在社会阅历和认知有提升之后。为什么现在网游那么吃香，大概也是因为单机被破解太多，不用购买、不用内购，开发者没有了那部分收入。可能就像千辛万苦养了个闺女……网游就不一样了，尤其是手游，充钱就是能变得更强，这或许会让人理解，玩游戏，真花钱。

  这是屁话了。尊重别人的劳动成果，那是应当的，鼓励支持劳动者，那也是值得赞扬的。“下次一定”也好，“上次给了”也好，都理应由他们自己发声。不能因为我没点这个菜，但是你手里拿着刀，把菜放我桌上，就认定这盘菜就得算到我头上。

  说起来我电脑的 Windows 系统也没有激活，所以右下角就一直提示我系统不是正版。我也不会像以前一样网上找个激活码或者破解器，因为我付不起那个正版的钱，所以我不需要享受激活后的便利。
