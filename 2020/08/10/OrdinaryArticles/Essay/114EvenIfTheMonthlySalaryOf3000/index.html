即使月薪三千，却也不能没有
发表于 2020-08-10 22:26:34| 更新于 2020-09-07 21:59:15|一般文字杂文随笔
字数总计:1.1k|阅读时长: 3 分钟

  我很疑惑，很多人无法想象月薪三千的生活，觉得现在这个时代，不可能还存在这样的工资水平。

  见笑了，望海涵。

  阿威离开工厂也有半年多了，前两天下班后，却也正好遇到了他，颇感意外，闲聊了半个多小时。聊聊近况，聊聊未来。

  阿威是产线作业员，可能也有一大部分人觉得不能理解，身强体壮，精力充沛的年轻人，为什么会想着进厂，而不是走南闯北，志在八方呢？现在遍地都是机会，遍地都是黄金，只要稍稍动点头脑，抓住机会，赚大钱那是手拿把攥的。

  不得不承认，进厂的年轻人的确还是很多，倒也不全都是不思进取，怕苦嫌累。可能因为学历不够，学习能力不强，不喜欢到处漂泊，已经习惯单纯机械工作、劳力工作等很多原因，他们还是进厂来了。

  我们聊着今年疫情原因，很多工厂都没有加班了，或者平常能加几天，但周末都不能加了，到哪儿都一样，有些厂还挺不住，垮了。一天八小时，一周八天，工作日加班 15 块左右一小时，一个月能加班满打满算 70 个小时，也就 1050，加上底薪一千七八百，现在一个月甚至拿不到三千块。

  他有问我现在一个月能拿多少，我不太好说，但他都说了，我不说就感觉不厚道，就只能说一个月五千左右，只拿底薪，没有加班。

  我很意外，他感觉很惊讶：这么多？然后问我：你是本科生吧？我含糊其词，说是。他继续说着话，言语中透露出羡慕。

  我时常怨恨自己，不思进取，985 毕业之后，到工厂做 web 开发，一个月只工作 174 个小时，没有加班，也没有多少钱。我一直劝解自己，学习一下，走出去，可以有更好的发展，也一直因为自己在工厂工作而羞愧。工厂 web 开发，没前途，没机遇，但这个阿威甚至都听不懂的名词，却依然被羡慕着，即便是打了折扣的收入。

  在外人看来，这可能就是格局的不同。在阿威看来，一个月能拿五千块就已经很多了。但是在名校毕业生看来，一个月三五万都可能显得不入流。我没有办法控制别人的看法，甚至都不能明确我自己有什么看法。正如阿威说的，虽然现在只有 3 千块一个月，但是不进厂打工，一个月一分钱都没有，但是吃住开销还是在那里，不花都不行。

  或许即便不是资本家，对于劳动力的“合理运用，创造财富”的做法，总是各有各有的。之前正如我对母亲抱怨，问她为什么一天天搞这么累，也多不到几个钱，回来躺都要躺半天。她跟我说：“打杂一天才 88，抢到个计件的活路，一天运气好可以搞到两三百块，还不是每天都有计件的活路干，你说要不要去抢活路？”

  我不知道她抢计件有抢得多辛苦，但回到出租屋要老爸捏腿捶背，洗了澡，吃了老爸做好的饭，可能一躺就睡着，就能睡到天亮，我看着的确也心疼。她告诉我，每天都只打杂，一个月就只有两千多，抢到几回计件，一个月就能能拿到三千多了。

  慢慢地，我能理解她的行为，但是，看着五十岁的父母为了一个月能多挣个五六百搞得精神疲劳，浑身疼痛。我还是很难接受。但是他们看着我，倒是觉得很意外，因为只有初中文化水平，出来打工十几二十年，他们信奉的准则非常简单：以痛换钱，无可厚非。

  我说我给她一个月三千块，回老家养养花种种菜，到街上打打麻将聊聊天过生活，她当然说好了。可是她身体力行的表明，她一个月想要的三千块，从来都不是从我这里能拿的三千块。
