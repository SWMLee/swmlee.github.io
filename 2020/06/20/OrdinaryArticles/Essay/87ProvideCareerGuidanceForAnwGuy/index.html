奇怪的角色：给小同事提供职业疏导？
发表于 2020-06-20 19:40:34| 更新于 2020-08-30 22:55:07|一般文字杂文随笔
字数总计:1.9k|阅读时长: 5 分钟

  小兄弟同事近来跟我讲诉了两个关于职业的心理困境的问题，不知道为什么，这两次我都跟他说了好多内容，甚至都把我自己折服：原来我的思想这么丰富而哲学的吗？简直就像人生导师一样。

一心只想搞 Java，就是不想学其他的

  几个月前，他问我的问题是：他一心只想搞 Java 相关的内容，无论是工作上还是未来规划上。但是领导有决策，有些内容不再用 java，然后使用 nodejs 进行开发。他觉得和自己希望偏差太多，心里很纠结，是按照公司规划学习新的自己不太熟悉的内容，还是换个地方、换个工作继续自己喜欢而又单纯的技术方向？

  我记得我当时说的内容大概是这么些：
  第一，你是个刚毕业的大学生，你熟悉、自以为很精通的东西，是真的就很厉害了吗？
  第二，真的是计划五年、十年都一直只用 java，其他一概不管的吗？

  年轻人大都气盛，第一点我以为他会谦虚一点，结果他真的觉得自己是足够优秀了，可以应对可能发生的大多数问题了，这我是很诧异的。不过我没有点破他，因为我知道我光说没有用，当他遇到问题解决不了的时候，自然会意识到原来自己还有不足。

  我更诧异的是他对第二点的态度，就是他觉得自己就是想搞 java 一辈子，可能像我这样的糙汉就觉得，技多不压身，多学习一点总是没有坏处的。但是他的观点就是：我坚持一门技术，到最后必然会非常精通，那么到那时，必然是中流砥柱了。我不知道是不是刚毕业刚工作时，都会对一门语言产生深厚的情感与喜爱，但是我个人觉得这种只想专注于一门开发语言的想法，实在不符合行业潮流，个人觉得吧，把职业生涯赌在单一一门开发语言上是很蠢的行为。

  所以，那次我说了很多，最后就给的很中庸的回答：如果下定决心只想搞 java，无法接受其他内容，想离职就离职完全没问题，不能刚出社会就被完全不喜欢的事物阻挠，（这么年轻就成为社畜肯定不好）如果只是觉得 java 心中地位非常重要，完全可以多学习扩展，但是心中永远都留着 java 的位置，并且时刻磨刀。这次学习 nodejs 就全当应付工作，甚至为了更好的淬炼技术自己在写个 java 版本，告诉领导，用 java 可以做得更好。说不定后续在本公司，你的能力能得到认可，公司决策偏向符合你的需要。

刚刚转正，觉得给得工资不够高

  今天他又来找我，带着另外一个问题来问我。他说最近几天心里想得很头痛，就是刚刚转正，觉得给得工资不够高，然后领导说 3 个月后看表现，评估这次转正工资是否可以进一步提升。因为后面这一句，他又非常想挣表现，但是觉得没有给到表现机会，很苦恼，天天都在想着心里好烦。

  这个问题我也很苦恼啊，我也没挣到钱啊，我的工资也是行业倒数啊，每次都拉队友后腿，在看统计数据的时候都很惭愧，所以我能给他什么建议呢？

  结果今天又硬生生跟他扯了一个小时。也不知道为什么，跟他扯了这两个着眼点。

  第一，你觉得低了，依据在哪里？

  因为他只是觉得低了，所以心里很苦恼。但是这个低了的标尺在哪儿呢？没有给我标尺，就这么一个结论。我跟他胡说：你觉得低了，那你需要找到这个比较的水平线在什么位置，你刚毕业不可能就早早和大学同学失联了，多问问他们现在什么水平的收入福利，并不一定直言问。还有就去找找看有没有网上一些统计资料，你的学历，你的资历，其他人大概什么水平的收入福利，取掉一些奇奇怪怪的峰谷点，得到一个标尺，这样你才能知道自己到底在一般大众什么水平。然后再看这个差距。

  当然我也有说这个差距的态度：如果你比水平低，可能就会觉得，凭什么我比其他人低，我要离职，完全 ok。第二，你跟其他人水平相当，可能就会觉得，我怎么可以跟他们一样，我比他们都厉害，我要离职，也完全 ok。第三，你比其他人高些，可能还会觉得，怎么只高了这么一点点，他们是白银，我怎么也得是个砖石啊，白金算什么话，想要离职，也没有问题。

  所以，第一个点就是找到这个标尺，比较这个差距，认清自己对这个差距的态度。

  第二，你的规划是什么？这里有没有辅助提升人生规划的东西？

  什么是表现？为什么要挣表现？怎么去挣表现？中间这个问题感觉很简单，因为有说可以涨工资，所以要去挣。但是第一个和第三个问题就不一定了。想要完成点项目、想要参与到设计实现，这算是表现，但是和领导同事的沟通，对待工作和周围人的态度等等乱七八糟的，这也算啊，不能觉得这个就不重要了。

  事实上，我跟他说的人生规划，其实就两个点：对于物质生活和情感生活的期待而已。因为作为社会人，这可能就是活着需要做的最重要的事情，无论狭义说美女与金钱也好，爱情与事业也好，就是这这么个问题。心情、同事情、友情、爱情、亲情这都是情感生活啊，工资，衣食住行，娱乐，这也都是物质生活啊，甚至还是有重合的。但是一般人都可能面对无法两手抓的问题。无法兼顾应该算是常态，所以在人生起步阶段，如何规划的这个侧重点就很重要了。

  我现在就是想挣钱，你的钱给得太少了，想离职就离职呗。但是我好喜欢隔壁同事小花啊，真想一直看着她，觉得为了还能看到小花钱少挣点无所谓就留下啊。

光想肯定没什么用

  所以整天想得头好痛，都不正确对待原因，光想着我好烦啊，我头好痛啊，没有意义。找到自己想要的，找到自身评价标尺，得出差距，正视态度，这算一步。人生规划侧重，环境辅助优化，这也算一步。找到这些东西，才是现在要做的。

  这种话怎么会从我的嘴巴里说出来？我也在想这个问题。或许假如当年我刚工作也有前辈跟我说这个问题，我就不会怀疑自己说的话了。
