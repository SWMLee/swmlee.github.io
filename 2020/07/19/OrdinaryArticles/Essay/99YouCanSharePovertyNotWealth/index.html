贫困同乐，富贵无言
发表于 2020-07-19 20:24:42| 更新于 2020-09-06 22:30:45|一般文字杂文随笔
字数总计:909|阅读时长: 2 分钟

  最近英雄联盟（简称 LOL）圈子有一个事件比较热闹，是关于某战队打野选手 ID 乐言（Leyan）与其成名前给予生活和事业很大帮助的，一个主玩影流之主劫的主播 ID 包子（bzzb）之间的。

  简单来讲，就是 02 年出生的乐言，在早些年只是一个“不务正业”但 LOL 游戏技术非常不错的小孩子，正好在 LOL 当时已经小有名气的主播包子遇到，觉得是“大腿”，便一起游戏，并在生活中给予一些帮助。后来觉得小伙子是个可造之材，通过自己各方努力，将其送到 LOL 职业联赛成为一名职业选手。还因为之前的战队对其发展不利，各方关系，送到世界冠军战队。从次级联赛，到了 LPL，加上之前战队拿到过世界冠军，流量比较大，据说因为其长相可爱，还有不少“妈妈粉”？

  然而就这么一个 18 岁的少年，从无人问津的网瘾少年，变成身价百(?)万的职业选手，再次和之前照顾和提携自己的老大哥包子双排的时候，态度却完全转变，据说从一口一个包哥，变成无话可说。 直播中一直不回应 bzzb 的交流，但却和疯狂互动，甚至唱歌。bzzb 像个傻子一样，不知所措，问对方怎么不说话啊，回复“有人……太吵了……影响训练”之类的。

  简而言之，此次的乐言事件，又是一个简单的“玩恩负义”的现实。尤其是被有心人找到了还未成名前乐言和包子之前的融洽，包子对乐言的各种帮助，今昔对比，瞬间把乐言推到了舆论风口，造就了各种有关的段子。例如乐言改了新 ID 叫 Leeya，因为“忘 N 复 E”。还有成名前说的话：“包哥能给我点个外卖吗？”、“包哥我能去你家过年吗？”被带入各种语境。此外“哦，XX，一个 XX 的 XX”也被疯狂改造。以前的“包哥乐弟”也变成了“包弟乐爷”，“如果乐言能够挺过去，在坐的各位都有责任”……当然，还有标题的那句：“贫困同乐，富贵无言”。

  其实我还是有些话想说的，没有必要，真的没有必要，人家才 18 岁啊，才刚刚成年啊。一个 18 岁的少年，成为职业选手，成为公众人物，日进斗金，那得是有多大压力啊。就不能体谅一下吗？别人在读书考大学的时候，人家就已经能够靠着大部分人说的不务正业的打游戏，不仅仅是养家活口，还能在短短几年之内，拥有很多人一辈子都赚不到的资源。这难道不是满满的正能量吗？普通人不好好读书，能取得他的成绩吗？何况人家也不是忘恩负义啊，时代变了，人不能再拿着旧时代的事情看问题了，过去的就过去吧。更何况一个努力在职业赛场的天才选手，和一个技术平平毫无人气的十八线小主播，也没什么话好说，圈不同，别硬融，难道不是吗？

  ——记者：“请问一下，作为一名未来可期的打野选手，能够取得今天的成绩，有什么话想对父母说的吗？”
  ——Leeya：“谁？”
  ——“您父母。”
  ——“哦，对两个养狗人有什么好说的。”

  （part1）
